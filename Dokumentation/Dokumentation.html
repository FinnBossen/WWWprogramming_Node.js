Ziel und Funktionsumfang der Anwendung

Wir möchtren ein Login für den Nutzer und dem Admin kreiren und somit unterschiedliche Rechte besitzen. Ziel ist es, dass Nutzer Bilder in der Galerie hochladen können und diese dann kommentieren, löschen und verändern können. Der Admin soll Beiträge anpassen können (Zeileiste, Tanzkurse) Einträge löschen und verändern können.

Zunächst wird die Login Seite geöffnet. Danach kann man sich asl Admin oder als normaler Nutzer registrieren. Direkt nach der Registration, muss man sich einloggen, um die Nutzerbezogenen Funktionen (Admin oder Nutzer) zu sehen. Man wird dann auf die Hauptseite weiter geleitet und kann sich nun durch die Website klicken. Die Funktionen für den Nutzer und dem Admin sind in Aufbau der Datenbank noch mal beschrieben.

Aufbau der Datenbank und Funktionen

Wir haben in der Datenbank mehrere Tabellen angelegt.

  • Zeitplan

    Admin: Der Admin kann Einträge hinzufügen, bearbeiten und löschen

    Nutzer: Der Nutzer kann den Zeitplan lediglich einsehen.

    Die Einträge werden nach Tage geordnet. Bei der letzten Leiste werden alle veränderbaren Attribute angezeigt, die dann vom Admin verändert oder gelöscht werden können.

  • Login

    • Man kann sich als Nutzer oder als Admin anmelden. Dies wird in der Datenbank gespeichert. Bei der Registrierung wird man sofort angemeldet. Außerdem wird man beim Login auf die Hauptseite weiter geleitet.

    Es wird eine Fehlermeldung angezeigt, wenn das Passwort oder der Namefalsch ist.

  • Galerie

    • Admin: Der Admin kann Beiträge verfassen aber auch alle anderen löschen.

    Nutzer: Der Nutzer kann Beiträge hinzufügen aber nur seinen eigenen ändern oder löschen.

    Das Bild kann gewechselt werden, ohne es zu löschen.

  • News

    • Admin: Der Admin kann neue Beiträge verfassen und alte Beiträge löschen.

    Nutzer: Der Nutzer kann die Beiträge ansehen.

    Nutzung von JS im Frontend

    Wir benutzen es um die Einträge zu bearbeiten. Der Nutzer kann kommentieren und Bilder hochladen und seinen Beitrag verändern oder löschen. Außerdem wird es im Kontaktformular verwendet. Der Administrator besitzt die gleichen Rechte, nur das dieser Webseiten interne Beiträge (Texte, Neuigkeiten, Zeitplan) und Beiträge von Nutzern bearbeiten und löschen kann.

    Überlegungen zum Progressive Enhancement

    Progressive Verbesserung beruht auf folgenden Prinzipien: Der Hauptinhalt sollte allen Browser verfügbar gemacht werden. semantische HTML-Auszeichnung Layout durch externe Stylesheets zusätzliche Interaktivität durch unobtrusive JavaScript Browserwahl und -Einstellungen der Nutzer werden respektiert Beispielsweise sollte ein Formular so erstellt werden, dass es sich auch ohne JavaScript absenden lässt und ohne CSS einen Sinn ergibt, auch wenn nicht alle von JavaScript-abhängigen Ausfüll-Hilfen funktionieren und es nicht „schick“ aussieht.

    Eine Lösung ist ein semantisches und responsives html. Damit lässt sich je nach Größe des Fensters eine passende Einstellung zum angucken der Seite finden. Außerdem benutzen wir externe css Dateien, um das html Dokument übersichtlicher zu halten. EIne weitere Lösung wäre, dass die Kontaktformulare so programmiert sind, dass diese auch abgeschickt werden, wenn javascript nicht funktioniert.

    Wer hat was umgesetzt?

    Im Allgemeinen saßen wir an allen Punkten gemeinsam. Es gab dann Aufteilungen in der Umsetzung, die wir aber auch gemeinsam besprochen haben.

    Welche externen Komponenten, Scripte, CSS etc werden verwendet?

    Wir benutzen Animate.css. Außerdem wurde die Website zuvor in einem anderen Studien Porjekt von Emanuel Maslama, Kristin Sönnichsen und Ann-Christin Frank erstellt. Hierfür benutzen wir die Daten und Bilder des Flensburger Tanzclubs mit Genehmigung. Die Html und Css Datein wurden eigenständig programmiert. Externe Quellen und Inspirationen wurden im Code dokumentiert.

    Warum werden sie verwendet?

    Animate.css wurde verwendet um die Seite attraktiver zu gestalten und das Ganze etwas lebendiger zu machen. Die Website wurde weiter entwickelt, da sie eine gute Grundlage für diese Hausarbeit bietet.
